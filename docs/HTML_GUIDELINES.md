# HTML Guidelines

## Naming

- kebab-case
- BEM

## Semantic

- header
- main
- section
- footer

## Accessibility

- button cho hành động
- a cho điều hướng

## Images

- decorative → aria-hidden="true"
- content → alt bắt buộc

## Scripts

- đặt cuối body hoặc dùng defer

## Comments

- Chia section bằng comment rõ ràng
